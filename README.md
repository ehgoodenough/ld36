# LD36 #

A game for [Ludum Dare 36](http://ludumdare.com).

![](http://i.imgur.com/wgHZg8J.gif)

### Getting Started ###

Have node v4 and npm v3.

    $ node --version
    $ npm --version

Build the game.

    $ npm install
    $ node build server

When reading the code, start at `source/index.js`.
